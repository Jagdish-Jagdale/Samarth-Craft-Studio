💡 Common Issues & Solutions

Status 200 but no message:

  • Template "ordercnfrm" not configured in WhatAPI dashboard
  • Template not approved by WhatsApp
  • Template parameter count mismatch (expects 7 params)
  • Phone number not registered in WhatsApp Business

Status 400:

  • Invalid phone number format
  • Template name incorrect
  • Message format doesn't match template

Status 429:

  • Rate limit exceeded
  • Too many messages sent in short time
  • Wait 30 minutes before testing again
